어시스트 모드의 도입 배경
공공 내부망은 네트워크 차단이 기본. 에이전트 도구(Claude Code·Cursor·Codex)는 외부 서버 통신이 전제라 쓸 수 없다. 이 현실이 4·5차시의 구조를 결정한다.
WHY왜 어시스트인가
내부망에서 에이전트 도구는 쓸 수 없다

Claude Code·Cursor·Codex는 외부 서버에 코드를 전송해 결과를 받는 구조. 공공 내부망은 외부 통신이 원천 차단되므로 사용 불가. 이 한계가 Stage B 설계의 출발점.

어시스트 정의4·5차시
AI가 코드를 "쓰고", 사람이 "옮기고 실행"한다

ChatGPT·Claude·Gemini 채팅창에 요구사항 입력 → 코드 수령 → 복사해서 내부망 실행. AI는 실행하지 않는다. 에러가 나면 로그를 다시 외부망에 보내 수정 요청.

6차시와의 대조
어시스트: 사람이 옮긴다 / 에이전트: AI가 직접 실행한다

같은 과제를 4차시(어시스트) → 6차시(에이전트) 두 번 수행하면서 차이를 체감하는 것이 Stage B→C 설계.

외부망·내부망 코드 이관 5단계
이 5단계가 4·5차시의 공통 기반. 처음엔 느리지만 10번 반복하면 15분 사이클로 체질화된다.
5 STEPS이관 절차
① 요구사항 전달 (외부망)

AI에 넘길 정보: 데이터 구조(컬럼명·타입·샘플 3행) + 기대 출력(차트 종류·형식) + 제약(오프라인 실행·금지 라이브러리·한글 폰트). 민감 데이터는 반드시 비식별화 후에만 전달.

② 코드 수령 + 검증

AI가 생성한 코드를 읽고 검증: 외부 통신(requests·urllib) 유무, 파일 경로 타당성, 내부망 라이브러리 가용성. 이해 안 가는 구간은 "이 3~5줄을 설명해줘"로 역질문.

③ 내부망 반입 + 실행

코드를 .py / .ipynb / .html로 저장 → USB 또는 승인 경로로 이동 → python script.py 실행. 결과 확인.

④ 에러 대응 사이클

에러 메시지 복사(내부 경로 비식별화) → 외부망 AI에 붙여넣기 → "수정해줘" → 수정 코드 수령 → 다시 ③. 체질화가 될 때까지 반복.

⑤ 아카이브 등록

완성 코드 + 요구사항 원문 + 에러 이력 + 산출물(PNG/HTML) 묶음 등록. 동일 업무 반복 시 재사용 가능.

PART E · pandas + AI 어시스트 패턴 — AI가 코드를 쓰는 4가지 문장형 · 10MIN
pandas 요청 문장의 4가지 전형
pandas 문법을 외우지 않는다. 요청 문장의 형태를 외우면 AI가 대부분을 쓴다.
4 PATTERNS문장형 4종
① 집계 문장

"[CSV]에서 [기준]별 [집계함수]를 구하고 [정렬]해서 [상위 N개]만 보여줘." → groupby + agg + sort_values + head. AI가 3줄 코드로 반환.

② 필터+집계

"[조건]을 만족하는 행만 남기고, [기준]별 [집계]를 해줘." → 조건 필터 후 groupby. "조건"은 일·부서·유형·범위 조합 가능.

③ 피벗·교차

"[행기준]×[열기준] 교차표로 [값]을 집계해줘." → pd.pivot_table. 엑셀 피벗과 동일한 구조를 자연어로 즉시.

④ 조인·통합

"[A.csv]와 [B.csv]를 [공통 키]로 합치고, [새 분석]을 해줘." → merge + 후속 분석. 부서코드·민원ID·접수일자 등 키가 명확해야.

포인트
"기준 · 집계함수 · 필터 · 정렬" 4요소를 문장에 담아라

이 4요소가 빠지면 AI가 임의 판단 → 틀린 결과. 명시만으로 정확도 90%↑.

SAMPLE실제 문장 → 코드 예시
예시 1 · 집계
"민원.csv에서 부서별 평균 처리일수를 구하고, 긴 순으로 정렬해서 상위 10개만 보여줘."

→ df.groupby('부서')['처리일수'].mean().sort_values(ascending=False).head(10)

예시 2 · 필터+집계
"2025년 3월~5월 민원 중 '환경'·'도로' 유형만 남기고 부서별 건수를 구해줘."

→ df[(df.접수일 >= '2025-03') & (df.접수일 < '2025-06') & (df.유형.isin(['환경','도로']))].groupby('부서').size()

예시 3 · 피벗
"부서×유형 교차표로 건수와 평균 처리일 두 값을 보여줘."

→ pd.pivot_table(df, index='부서', columns='유형', values='처리일', aggfunc=['count','mean'])

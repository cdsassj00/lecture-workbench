AXIS 1·2·3수정·실행·에러
AXIS 01 · 파일 수정
"여기에 이렇게 붙여" → 직접 편집

어시스트는 코드 블록을 보여주고 사용자가 복사/저장. 에이전트는 diff를 보여주고 승인만 받아서 파일에 직접 반영. 공공 환경에서는 diff 단계가 곧 감사로그.

AXIS 02 · 실행
터미널이 에이전트의 손이 된다

pytest·python main.py·npm run 을 에이전트가 직접 실행 → 결과를 읽고 다음 판단. 사람은 결과와 로그만 본다.

AXIS 03 · 에러 대응
트레이스백을 읽고 스스로 재시도

어시스트: 사람이 에러를 복사해 다시 질문. 에이전트: 에러 → 원인 가설 → 재수정 → 재실행을 한 루프 안에서 처리.

AXIS 4·5·6다파일·기억·반복
AXIS 04 · 다파일
프로젝트 트리 전체를 맥락으로

10개 파일에 걸친 리팩토링은 어시스트로 불가능. 에이전트는 ls·grep·read를 내장해 트리를 읽고 관련 파일을 스스로 연다.

AXIS 05 · 장기 기억
메모리·파일로 컨텍스트 유지

세션이 바뀌어도 CLAUDE.md·AGENTS.md·메모리 파일이 남아 있으면 "이 프로젝트의 규칙"을 다시 설명하지 않아도 된다.

AXIS 06 · 반복 지시
Rules에 박아놓고 잊는다

"한국어로", "파일명은 {날짜}_{용도}" — 이런 반복 지시를 매 세션마다 타이핑하지 않는다.

에이전트 전환 시점 판단 기준
모든 업무를 에이전트로 옮기는 것이 목적이 아니다. 전환 신호를 기준으로 분기한다.
5 SIGNALS에이전트 전환 신호
SIGNAL 01
같은 지시를 3번 이상 반복하고 있다

"한국어로", "주석은 쉽게", "파일명 규칙" — 반복이 있으면 Rules 자산화 시점.

SIGNAL 02
파일이 5개 이상 엮여 있다

어시스트는 한 파일 한 덩어리. 파일 간 의존성이 있으면 에이전트가 트리 전체를 읽어야 한다.

SIGNAL 03
실행·에러 루프가 5번 이상 돈다

"코드 복사 → 실행 → 에러 → 재질문"을 다섯 번 이상 하고 있다면 에이전트로 자동화할 신호.

SIGNAL 04·05
④ 다음 사람이 재현해야 한다 / ⑤ 감사·검수가 필요하다

재현성·감사로그가 필요한 공공 업무는 에이전트 + AGENTS.md + 로그 파일의 조합이 가장 깔끔하다.

설계서를 쓰는 데 드는 시간이 과제 소요 시간보다 길면 배보다 배꼽. 아래 경우는 어시스트(ChatGPT·Claude·Gemini 채팅)로 가는 게 낫다.

Cursor의 Rules·Composer 활용
Cursor의 핵심은 규칙과 Composer다. 두 기능을 함께 쓰면 IDE 에이전트의 장점이 완성된다.
.CURSOR/RULES규칙 파일
규칙은 주제별 .md 파일로 분리

CLAUDE.md가 한 장이라면 Cursor는 .cursor/rules/ 폴더 안 여러 파일. 파일마다 "언제 적용할지"를 frontmatter로 지정한다.

// .cursor/rules/style-korean.md --- description: 한국어 공공기관 스타일 globs: ["**/*.py", "**/*.md"] alwaysApply: true --- 당신은 공공기관 실무 개발자입니다. - 주석은 한국어로 작성 - 파일명: {YYYYMMDD}_{용도}.py - 민감 정보는 로그에 남기지 않음 // .cursor/rules/security-public.md --- description: 공공 보안 가이드 globs: ["**/*.py"] --- - API 키는 .env, 하드코딩 금지 - 외부망 호출은 별도 모듈 - 주민번호·이름·연락처 로그 출력 금지
TIP
Rules는 주제별 3~5장으로 쪼갠다

한 파일에 몰아넣지 말고 스타일·보안·검증·프로젝트 규칙을 분리.

COMPOSER다중 탭 작업
Ctrl+I — 여러 파일을 한 번에 여는 "설계서 모드"

Composer는 에이전트에게 여러 파일 동시 수정을 시킨다. 새 기능 추가 시 model.py·views.py·tests/*.py를 한꺼번에 업데이트.

USE CASE
리팩토링 · 아키텍처 변경

클래스 이름 변경·API 시그니처 변경 — 여러 파일 일관성 유지.

CHECKPOINT
변경 전 스냅샷 자동 생성

실수했을 때 한 번에 되돌리기 가능. 공공 업무에서 안심 장치.

PITFALL
▸ Composer로 너무 큰 범위 한 번에

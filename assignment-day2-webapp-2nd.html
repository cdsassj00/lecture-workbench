Assignment Brief · For Information Only

01주제

Lovable에 프론트엔드 + 백엔드(DB · API Edge Function · 인증 · Storage) 기능이 있는 사이트를 하나, 자유 주제로 제작합니다.

문서 편집기부터 게임, 디자인물까지 자유롭게 상상력으로 창작해주세요.

02개발·제출 절차

STEP 1
Lovable 제작
자유 주제로 사이트를 기획·제작합니다.
STEP 2
Publish · GitHub 연동
배포(Publish)까지 완료해 URL을 확보하고, GitHub Integration으로 코드를 내보내 GitHub 주소도 확보합니다.
STEP 3
구글시트 제출
구글시트에 소속 · 이름 · Lovable URL · GitHub 코드 URL을 입력해 제출합니다.

03제출 항목 (구글시트 1행)

  • ① 소속
  • ② 이름
  • ③ Lovable URL · 배포(Publish)된 사이트 주소
  • ④ GitHub URL · Integration으로 내보낸 코드 저장소 주소
사용 가이드 · 평가 기준

• Lovable 크레딧은 1일 5개 한도입니다. 하루에 끝내지 못해도 OK.

• 미완성 = 감점 없음 — 크레딧 제한을 감안한 평가 정책입니다.

• 단, 기획력에 초점을 맞춰 평가합니다.

Note · 참고

본 페이지는 과제 안내문입니다. 실제 제출은 위 절차에 따라 구글시트에 한 행을 추가하는 방식으로 진행해주세요.

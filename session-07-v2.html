6주차 산출물 점검 — 핵심 개념과 작동 원리
각자 만든 MVP를 짧게 시연한다. 막힌 지점을 모아 이번 주차 학습 메뉴에 반영.
CONCEPT · 핵심 개념
6주차 산출물 점검의 핵심 한 줄
강사 작성 영역 이 도구·기법이 무엇이고 왜 다루는가를 1~2문단으로 정리.
  • 핵심 정의 한 줄
  • 왜 이 자리에 들어가는가 (앞 PART와의 연결)
  • 거점리더가 알아야 할 최소 원리
팁 — 수강생 수준에 맞춰 1) 비유 1개 2) 1문장 정의 3) 시각 자료 1개로 압축한다.
MECHANICS · 작동 원리
이렇게 동작한다
강사 작성 영역 도구 또는 기법의 작동 원리·구성요소·데이터 흐름을 한 장으로 정리.
  • 구성요소 3~5개
  • 입력 → 처리 → 출력 흐름
  • 거점리더가 자주 만날 함정 1~2개
팁 — 수식·코드는 최소화. 박스·화살표 다이어그램으로 한 장 안에 끝낸다.

PART 01 · PRACTICE · 실습 시나리오
6주차 산출물 점검 — 손을 움직이는 시간
이 슬라이드는 수강생이 직접 따라하거나 도전하는 실습 단위 한 장. 입력·과정·산출물·검증 기준을 명시한다.
SCENARIO · 시나리오
오늘 만들 것을 한 줄로
강사 작성 영역 실습 미션을 1문장으로 정의. 누가, 무엇을, 어디까지 만드는가.
  • 대상 사용자(거점리더 자신/동료/시민)
  • 입력 데이터·자료
  • 완성 기준 1~2개
팁 — '만들 산출물 파일 형식'이 정해져야 시간 계산이 가능하다.
STEPS · 손길 단계
3~5단계로 끊어 쓴다
강사 작성 영역 실제 손길 단계 3~5개. 각 단계는 한 줄로 끊는다.
  • STEP 1 — 입력 준비
  • STEP 2 — 도구 호출 / 프롬프트
  • STEP 3 — 결과 검증
  • STEP 4 — 정리 / 공유
팁 — 쉬는 시간 빼고 25~40분 안에 끝나는 단위.
CHECK · 검증 / 산출물
어디까지 되면 통과인가
강사 작성 영역 통과 기준과 산출물 위치를 명시.
  • 완성 기준(예: 동료가 같은 입력으로 같은 결과)
  • 산출물 저장 위치(공용 폴더/GitHub Org)
  • 흔한 실패 패턴 1~2개
팁 — 통과 기준이 1줄로 안 적히면 실습이 흐릿해진다.

API / MCP 연동 실습 — 핵심 개념과 작동 원리
외부 API와 MCP 서버를 붙인다. 거점 미션을 다른 시스템과 연결하는 표준 인터페이스.
CONCEPT · 핵심 개념
API / MCP 연동 실습의 핵심 한 줄
강사 작성 영역 이 도구·기법이 무엇이고 왜 다루는가를 1~2문단으로 정리.
  • 핵심 정의 한 줄
  • 왜 이 자리에 들어가는가 (앞 PART와의 연결)
  • 거점리더가 알아야 할 최소 원리
팁 — 수강생 수준에 맞춰 1) 비유 1개 2) 1문장 정의 3) 시각 자료 1개로 압축한다.
MECHANICS · 작동 원리
이렇게 동작한다
강사 작성 영역 도구 또는 기법의 작동 원리·구성요소·데이터 흐름을 한 장으로 정리.
  • 구성요소 3~5개
  • 입력 → 처리 → 출력 흐름
  • 거점리더가 자주 만날 함정 1~2개
팁 — 수식·코드는 최소화. 박스·화살표 다이어그램으로 한 장 안에 끝낸다.

SCENARIO · 시나리오
오늘 만들 것을 한 줄로
강사 작성 영역 실습 미션을 1문장으로 정의. 누가, 무엇을, 어디까지 만드는가.
  • 대상 사용자(거점리더 자신/동료/시민)
  • 입력 데이터·자료
  • 완성 기준 1~2개
팁 — '만들 산출물 파일 형식'이 정해져야 시간 계산이 가능하다.
STEPS · 손길 단계
3~5단계로 끊어 쓴다
강사 작성 영역 실제 손길 단계 3~5개. 각 단계는 한 줄로 끊는다.
  • STEP 1 — 입력 준비
  • STEP 2 — 도구 호출 / 프롬프트
  • STEP 3 — 결과 검증
  • STEP 4 — 정리 / 공유
팁 — 쉬는 시간 빼고 25~40분 안에 끝나는 단위.
CHECK · 검증 / 산출물
어디까지 되면 통과인가
강사 작성 영역 통과 기준과 산출물 위치를 명시.
  • 완성 기준(예: 동료가 같은 입력으로 같은 결과)
  • 산출물 저장 위치(공용 폴더/GitHub Org)
  • 흔한 실패 패턴 1~2개
팁 — 통과 기준이 1줄로 안 적히면 실습이 흐릿해진다.

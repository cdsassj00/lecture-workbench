01주제

자신이 속한 기관 또는 팀에 도움이 될 만한 웹앱을 개발하여 공유해주세요.

Pain Point에 따라 블로그·홈페이지 형태도 가능합니다.

02개발 절차

STEP 1
Google Stitch
필요한 앱의 화면과 기능을 디자인·기획합니다.
STEP 2
AI Studio · Build
Google AI Studio 내 Build 기능으로 실제 앱을 개발합니다.
STEP 3
배포 · 링크 생성
대외에서 접근 가능한 링크를 생성·배포합니다. AI Studio 내부 기능 또는 GitHub 연동 후 Vercel 같은 바이브코딩 플랫폼 활용 가능, 그 외 자율 방식도 가능합니다.

03최종 제출물

구글독스 보고서 1부 — 아래 내용을 포함해주세요.

  • ① 제3자가 접근 가능한 웹상의 주소 링크
  • ② 소속팀·기관 내 Pain Point 정의 및 설명
  • ③ 그에 대한 해결책으로서 앱에 대한 소개와 주요 기능

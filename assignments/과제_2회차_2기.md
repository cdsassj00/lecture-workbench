# 2회차 과제 안내 · 2기

## 1. 주제
**Lovable**에서 프론트엔드 + 백엔드 기능을 모두 갖춘 **사이트 1개**를 자유 주제로 제작하여 배포해주세요.

- 자유 주제: 문서 편집기, 게임, 디자인 도구 등 **상상력 기반 창작** 권장
- 백엔드 포함 범위: **DB · API Edge Function · 인증 · Storage**

## 2. 개발 절차
1. **Lovable**에서 사이트 기획·제작
   - DB / Edge Function / 인증 / Storage 중 필요한 기능 자유 구성
2. **GitHub Integration** 연동하여 코드 내보내기
3. **Publish**로 배포 URL 확보

## 3. 최종 제출물 (구글시트 1행 입력)
| 항목 | 내용 |
| --- | --- |
| 소속 | 기관·팀명 |
| 이름 | 본인 성명 |
| Lovable URL | 배포된 사이트 주소 |
| GitHub URL | Integration으로 내보낸 저장소 주소 |

## 4. 사용 가이드 (운영 안내)
> 💡 **크레딧 운용 팁**
> - Lovable 크레딧은 **1일 5개** 한도 → 하루에 다 못 끝내도 OK
> - **미완성도 감점 없음** (크레딧 제약을 감안한 평가)
> - 단, **기획력**에 초점을 맞춰 평가
> - 완벽한 구현보다 "무엇을 왜 만들고 싶었는가"가 핵심

## 5. 체크리스트
- [ ] Lovable에서 자유 주제 기획
- [ ] 프론트엔드 + 백엔드(DB·인증·Storage·Edge Function 중 1개 이상) 구성
- [ ] Publish로 배포 URL 확보
- [ ] GitHub Integration 연결 및 저장소 URL 확보
- [ ] 구글시트에 소속·이름·Lovable URL·GitHub URL 입력
